# Odin Recipes

A recipe website project created to practice embedding images and linking multiple nested pages together as part of The Odin Project curriculum.

## 📝 What I Practiced

- **Image Embedding:** Using `<img>` tags to display local images located inside a dedicated asset folder (`./images/`).
- **Multi-Page Navigation:** Linking the main index page to multiple nested HTML files inside the recipes directory.
- **Asset Organization:** Separating content (`recipes/`) and visual assets (`images/`) into their own distinct directories for a cleaner project structure.

## 📁 File Structure

```text
.
├── README.md
├── images
│   ├── banana-pancakes.jpg
│   ├── chicken-fried-rice.jpg
│   └── grilled-cheese-sandwich.jpg
├── index.html
└── recipes
    ├── banana-pancakes.html
    ├── chicken-fried-rice.html
    └── grilled-cheese-sandwich.html
```

## 🚀 How to View

Simply open `index.html` in any web browser.
